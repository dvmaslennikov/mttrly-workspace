# Twitter Ops — mttrly

Цель: вести полезно-развлекательный, профессиональный X-аккаунт без пушинга, накапливая доверие и экспертность.

## Тон
- Спокойно-пофигистичный, очень профессиональный
- Коротко, ёмко, с сухой иронией
- Без агрессивных продаж, без кликбейта

## Артефакты
- `daily-process.md` — ежедневный процесс (что/когда делать)
- `query-fanout.md` — библиотека поисковых интентов и запросов
- `scoring-and-routing.md` — как выбирать, на что отвечать
- `response-policy.md` — правила ответов и ограничений
- `daily-template.md` — шаблон ежедневного отчёта/пака
